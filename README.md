# BluetoothHelper

安卓蓝牙调试工具

这是一个蓝牙助手，用于与其它蓝牙设备调试蓝牙连接的工具。是基于Android SDK进行蓝牙连接的。

这是一个Eclipse工程,不适合使用Android Studio。
